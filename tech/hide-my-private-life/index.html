깃허브 블로그에 나만의 비밀 일기장 만드는 법 (카테고리 숨기기)

May 27, 2026

## 1. 남들에게 보여주기 부끄러운 솔직한 기록들

기술 블로그를 운영하다 보면, 가끔은 정돈된 개발 지식 말고 날것 그대로의 감정이나 찌질하고 솔직한 고민들을 적어두고 싶을 때가 있습니다. 하지만 전 세계 누구나 볼 수 있는 인터넷 공간에 내 깊은 속마음을 모두 공개하기는 너무 창피하고 부담스럽죠. 그래서 기술 블로그 안에 나 혼자만 볼 수 있는 ‘비밀 일기장’ 공간을 숨겨놓는 방법을 적용해 보았습니다.

## 2. 마법의 주문: draft = true

Hugo 블로그에서는 글을 인터넷에 올릴지 말지를 아주 간단한 단어 하나로 결정할 수 있습니다. 바로 글 맨 위에 있는 설정창의 draft 속성입니다.

평소에는 글을 다 쓰고 나면 인터넷에 올라가도록 draft = false라고 적지만, 남몰래 쓴 일기장 파일(content/life/내일기.md)에는 이 부분을 아래처럼 바꿔줍니다.

draft = true

이렇게 true(초안 상태)로 바꿔서 저장하면, 블로그를 조립하는 로봇이 “아, 이 글은 아직 인터넷에 올리면 안 되는 임시 글이구나!” 하고 쏙 빼고 웹사이트를 만들어 줍니다. 내 하드디스크에는 파일이 고스란히 남아있지만, 인터넷 세상에는 아예 흔적조차 남지 않게 되는 것이죠.

## 3. 블로그 메뉴판에서 간판 내리기

글은 숨겼는데 막상 블로그 화면 우측 상단에 ‘삶의 기록’이라는 메뉴 버튼이 그대로 남아있으면 이상하겠죠? 메뉴 버튼도 함께 숨겨주어야 합니다.

블로그의 전체 설정을 관리하는 hugo.toml 파일을 열고, ‘삶의 기록’ 메뉴를 만들었던 코드 줄 맨 앞에 해시 기호(#)를 붙여줍니다.

[[menus.main]]

name = “삶의 기록”

url = “/life/”

weight = 20

컴퓨터 세상에서 # 기호는 “이 코드는 무시하고 넘어가라"는 뜻입니다. 이렇게 주석 처리를 해두면 블로그 화면에서 삶의 기록 메뉴 버튼이 감쪽같이 사라집니다.

## 4. 나 혼자 비밀글을 확인하고 싶을 땐?

인터넷에는 안 올라갔지만, 내 컴퓨터 모니터에서는 실제 블로그 화면처럼 예쁘게 일기를 읽고 싶을 수 있습니다. 그럴 땐 터미널에 평소 치던 명령어 뒤에 대문자 -D 하나만 더 붙여주면 됩니다.

hugo server -D

이 명령어는 “인터넷에 올리지 않으려고 숨겨둔 초안(Draft) 글들까지 전부 합쳐서 내 모니터에만 띄워줘!“라는 뜻입니다.

## 5. 마치며

어려운 보안 시스템이나 로그인 기능을 만들지 않아도, 텍스트 몇 글자만 바꾸면 완벽한 나만의 비밀 공간이 생겨납니다. 남들의 시선을 신경 쓰지 않고 내 솔직한 생각들을 차곡차곡 모아둘 수 있는 든든한 다락방이 하나 생긴 것 같아 마음이 편안해집니다.
