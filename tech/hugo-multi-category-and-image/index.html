블로그 카테고리(tech/life) 나누고 이미지 넣는 방법

May 26, 2026

1. 카테고리를 나눈 이유

블로그에 글을 쓰다 보니, 개발을 공부한 ‘기술 기록’과 평범한 일상을 적은 ‘삶의 기록’이 한곳에 섞여 있는 게 보기 불편했습니다. 이 두 가지를 깔끔하게 분리하고, 글 중간에 사진도 예쁘게 넣고 싶어서 블로그 구조를 바꿔보았습니다.

2. 폴더를 나누면 카테고리가 됩니다

Hugo 블로그는 폴더를 나누는 것만으로도 인터넷 주소가 분리됩니다.

  • content/tech/: 기술 관련 글을 모아두는 곳
  • content/life/: 일상 에세이를 모아두는 곳

3. 상단 메뉴바에 버튼 만들기

카테고리를 나눴으니 사람들이 클릭할 수 있게 메뉴 버튼을 만들어야 합니다. 블로그 전체 설정을 담당하는 hugo.toml 파일 맨 밑에 아래 코드를 추가했습니다.

[[menus.main]] name = “기술 기록” url = “/tech/” weight = 10

[[menus.main]] name = “삶의 기록” url = “/life/” weight = 20

4. 글 안에 사진 깨짐 없이 넣기

글 속에 사진을 넣으려면 내 컴퓨터 static/images/ 폴더에 사진을 먼저 저장하면 됩니다. 그리고 글을 쓸 때는 아래처럼 적어줍니다.

여기서 가장 중요한 꿀팁은, 주소를 적을 때 static이라는 글자는 빼고 /images/부터 적어야 나중에 웹사이트에서 사진이 엑스박스 없이 짠 하고 잘 나옵니다.

5. 마무리

처음엔 막막했는데, 내 손으로 직접 폴더를 나누고 메뉴를 만들어보니 블로그가 한결 다채로워졌습니다. 이제 이 공간에 저만의 개발 공부와 삶의 조각들을 차곡차곡 쌓아갈 생각에 설렙니다.
